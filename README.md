##music-artist-recommender
Fetches real data from an external API and presents it in a useful, well-designed interface

##What your app does:
This app will allow users to search for music artists and then view the artist's information in an interactive and visually engaging layout.

##Which API you used (with link)
iTunes Search API
https://itunes.apple.com/search
